# TinyAtom Atom Submissions

TinyAtom is a desktop app for building and using small, focused apps called atoms. This repository is the
review queue for atoms being published to the public TinyAtom marketplace.

## How to submit an atom

You do not submit here by hand. Build your atom in the TinyAtom desktop app, then use the app's option to
publish it to the public marketplace. The app prepares your atom and submits it here for review
automatically.

A maintainer reviews each submission. Once yours is accepted, your atom becomes available in the TinyAtom
marketplace.

## Updating an atom you already published

Raise your atom's version in the app and publish it again.
